# NLW Esports
![image](imgsREADME/demo.png)

## O seu duo 💏 está aqui! 

O NLW Esports é uma plataforma web e mobile de anúncios de gamers a procura de alguém para jogar nos horários disponiveis.

## Tecnologias utilizadas na construção 🤖

- React
- Node.js
- React Native
- Expo
- Typescript
- Express
- MySQL
- Prisma
- Tailwind
- PhosporIcons
- Radix UI
- OAuth2 (Discord)
- JWT
